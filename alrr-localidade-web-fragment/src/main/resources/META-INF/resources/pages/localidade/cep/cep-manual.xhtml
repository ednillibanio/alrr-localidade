#Cadastro de Cep

Esta funcionalidade é responsável por manter os cep(s) na base local. Todo cep cadastrado na base local é consultado numa base pública de cep(s) mantida na internet. Para mais informações de como o Cep é cadastrado, consulte o processo de Novo Cep.

Ações

Novo Cep

Esta ação cadastra um novo cep na base de dados local. Para cadastrar um novo cep, (1) o usuário informa o cep no formulário de cadastro, (2) o sistema consulta-o na internet e (3) retorna os campos (logradouro, bairro se houver, município e Uf) preenchidos automaticamente.

Quando um cep é cadastrado na base local, o sistema também cadastra o Bairro e o Município se ainda não estiverem cadastrados. Para vizualiza-los, acesse a funcionalidade de Cadastro de Bairro e Cadastro de Municípios.

Geralmente os municípios já estão todos cadastrados. Pois uma rotina de importação de municípios do IBGE já foi executada. Para mais informações consulte o manual de Cadastro de Municípios.

Editar

Esta ação permite que os dados (logradouro e bairro) do cep sejam alterados.

Detalhes

Esta ação exibe os dados do cep. Não é possível editar os dados.

Excluir

Esta ação exclui o cep da base de dados local após confirmação.

Validações

Regras / Restrições

  • O campo Cep, Município e Uf não podem ser alterados;
  • Para cadastrar um novo cep, é necessário que o sistema tenha acesso a internet e que a base de cep(s) na web esteja online;
  • O sistema não permite o mesmo cep cadastado mais de uma vez;
  • Todos os campos marcados com "*" são obrigatórios.
