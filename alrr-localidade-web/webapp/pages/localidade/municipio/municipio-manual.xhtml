#Cadastro de Município

A funcionalidade Cadastro de Município é responsável pela manutenção dos municípios que são usados por várias outras funcionalidades. É um cadastro único compartilhado com outros sistemas. Dessa maneira mantém-se uma base consistente, evitando-se a redundância de dados e facilitando a manutenção.

Ações

Importar Municípios IBGE

Esta ação é responsável por importar todos os municípios da base do IBGE para a base local do sistema. É uma tarefa que facilita o cadastro dos municípios e mantém os dados consistentes. Segue o padrão de nomes, nomenclaturas e códigos do IBGE.

Pode ser executada várias vezes. Na primeira vez, o sistema importa todos os municípos do IBGE. Nas execuções seguintes, o sistema cadastra apenas os novos municípios criados pelo IBGE e que ainda não estão na base local do sistema. Além disso, o sistema atualiza os dados dos municípios que sofreram alterações no cadastro do IBGE.

Novo Município

Esta ação cadastra um novo município na base local. Ela pode ser usada quando o usuário deseja cadastrar apenas um ou uma pequena quantidade de municípios. Deve ser usada em exceções, quando o Importar Municípios IBGE está fora do ou ou uma pequena quantidade. a ação de excluir um registro. Não é recomendado excluir registros. Por isso, a ação de desativar, altera a situação do feriado para inativo e consequentemente para de ser exibido nas consultas de busca em outras funcionalidades. Por exemplo, um feriado que está inativo deixa de ser exibido num calendário.

Validações

  • O valor do campo Dia deve ser entre 1 e 31.

Restrições

  • O sistema não permite mais de um município cadastado com o mesmo Nome e UF.
  • O campo Ibge Cod. é único.
  • Todos os campos marcados com "*" são obrigatórios.
  • Registro que a situação seja Inativa não aparece nos componentes de seleção de valores. Por exemplo, se Boa Vista estiver inativa, ela não será exibida no componente de seleção de Naturalidade de Pessoa Física.
